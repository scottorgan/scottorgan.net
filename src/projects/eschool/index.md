  <h3 style="padding-top: 10px;">eSchool Extension for Chrome (for Arkansas K12 Schools)</h3>
</div>

<p align="center" style="padding: 20px;"><img src="/images/eschool_screenshot.png" alt="Screenshot" width="66%"></p>

<p>
    A simple extension that configures Google Chrome for use with eSchool.
</p>

<ul>
    <li>Puts a convenient eSchool shortcut on the toolbar.</li>
    <li>Removes the requirement to "Load Unsafe Scripts" when opening Cognos.</li>
    <li>Configures Chrome to always allow pop-ups from eSchool.</li>
</ul>

<br />

:::tip NEW for 1.1:
Right-click the eSchool toolbar icon for quick access to other related websites, including easier access to Cognos.

<small><strong>Note:</strong> For faster access to Cognos, configure your DSN setting by right-clicking the eSchool toolbar icon and selecting 'Options'.</small>
:::
